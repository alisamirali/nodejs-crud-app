## CRUD-APP

### Simple CRUD App using Node.js, Express.js, and MongoDB.

---

- GET GetProducts

`http://localhost:3000/api/products`

- POST CreateProduct
 
`http://localhost:3000/api/products`

- GET GetProduct
 
`http://localhost:3000/api/products/:id`

- PUT UpdateProduct
 
`http://localhost:3000/api/products/:id`

- DELETE DeleteProduct

`http://localhost:3000/api/products/:id`

- GET TestServer
 
`http://localhost:3000/`
